## structure
- backend = laravel
- frontend = nextjs
- db = sqlite

## how to run
- clone git **main** branch to local
- make sure to install all dependencies (php, laravel, nodejs, nextjs)
- run
```
php artisan migrate:refresh --seed
php artisan serve
npm run dev
```

- akses di browser di halaman http://localhost:3000
